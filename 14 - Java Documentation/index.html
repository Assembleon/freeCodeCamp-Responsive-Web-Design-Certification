Introduction

Java is a high-level, class-based, object-oriented programming language that is designed to have as few implementation dependencies as possible. It is a general-purpose programming language intended to let application developers write once, run anywhere (WORA), meaning that compiled Java code can run on all platforms that support Java without the need for recompilation.

What you should already know

This guide assumes you have the following basic background:

  • A general understanding of the Internet and the World Wide Web (WWW).
  • Good working knowledge of HyperText Markup Language (HTML).
  • Some programming experience. If you are new to programming, try one of the tutorials linked on the main page about Java.
Java vs. JavaScript

While Java and JavaScript share similar names and are both widely used in web development, they are fundamentally different in several ways. Java is a statically typed, compiled language used primarily for backend development, while JavaScript is a dynamically typed, interpreted language used mainly for frontend development.

Hello World

To get started with writing Java, here is your first "Hello World" Java code:

public class HelloWorld {
  public static void main(String[] args) {
    System.out.println("Hello, World!");
  }
}

Save this code in a file named HelloWorld.java, then compile and run it using the Java compiler and runtime.

Variables

In Java, variables are containers that hold data that can be changed during the execution of a program. Here is how you can declare variables:

int myNumber = 42;
String myString = "Hello, World!";
Data Types

Java supports several data types including:

  • int - for integers
  • double - for floating-point numbers
  • boolean - for true/false values
  • char - for single characters
  • String - for a sequence of characters
Control Flow Statements

Control flow statements in Java include if, for, while, and switch statements. Here is an example of an if statement:

if (condition) {
  System.out.println("Condition is true");
} else {
  System.out.println("Condition is false");
}
Methods

Methods are blocks of code that perform a specific task. Here is how you can define a method in Java:

public int addNumbers(int a, int b) {
  return a + b;
}
Classes and Objects

Classes are blueprints for creating objects. Here is an example of a simple class:

public class Dog {
  String name;
  int age;
  public void bark() {
    System.out.println("Woof!");
  }
}
Inheritance

Inheritance allows one class to inherit the fields and methods of another class. Here is an example:

public class Animal {
  public void eat() {
    System.out.println("This animal eats");
  }
}
public class Dog extends Animal {
  public void bark() {
    System.out.println("Woof!");
  }
}
